LifeLink backend only

LifeLink is an emergency first blood donor coordination platform designed to connect nearby donors with patients in urgent need.

 Features
- Donor Registration
- Blood Request Creation
- Real-time Donor Matching (by blood group)
- MongoDB Database Integration

 Tech Stack
- Node.js
- Express.js
- MongoDB (Atlas)

 How to Run

1. Clone the repo:
   git clone https://github.com/Nephilimking/LifeLink-backend.git

2. Install dependencies:
   npm install

3. Create .env file:
   MONGO_URI=your_mongodb_uri

4. Run server:
   node server.js
